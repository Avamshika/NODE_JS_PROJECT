# Smart Student Profile - ReactJS

This project is part of a **NodeJS Activity**. It is a dynamic profile application built with ReactJS that allows users to view and update student information.

## Features
* **Default Data Loading**: Automatically displays initial student details.
* **Interactive Profile**: View student name, roll number, and bio.
* **Edit Functionality**: Toggle between "View" and "Edit" modes.
* **State Management**: Updates and saves details using React Hooks (`useState`).

## How to Run
1. Clone the repo.
2. Run `npm install`.
3. Run `npm start`.
